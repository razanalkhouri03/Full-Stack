# Fullstack App (Next.js + NestJS + MongoDB)

## Voraussetzungen

- Docker
- Docker Compose

## Projekt starten

Im Projektordner folgenden Befehl ausführen:

```
docker-compose up --build
```

## Anwendung erreichen

Nach dem Start ist die App erreichbar unter:

http://localhost:3000

## Container stoppen
```
docker-compose down
```
